# Adobe MCP gateway servers

Adobe provides the following MCP servers that you can connect to MCP clients:

<Product-Card slots="icon, heading, text, buttons" repeat="2" />

![Adobe Analytics icon](assets/platform_service.svg)

### Adobe Analytics

Run Adobe Analytics reporting queries through MCP and return results aligned with Analysis Workspace reporting. Use it to fetch metrics, dimensions, segments, and time-based breakdowns for analysis and automation in your LLM workflow.

* [Learn more](aa/index.md)
* [View available tools](aa/reference.md)

![Customer Journey Analytics icon](assets/platform_service.svg)

### Customer Journey Analytics

Run Customer Journey Analytics reporting queries through MCP and return results aligned with Analysis Workspace reporting on unified, cross-channel customer journey data. Use it to retrieve metrics, dimensions, components based on derived fields, and segment-based breakdowns for analysis and automation in your LLM workflow.

* [Learn more](cja/index.md)
* [View available tools](cja/reference.md)
